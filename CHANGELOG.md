# 專案版本更新紀錄 (CHANGELOG)

## [V5.8] - 2026-04-12 (CLEAR ALL & LAYOUT PROTECT)
### ✨ 新增 (Features)
- 實裝「清空整月預約」終極清除鈕：提供二重強制防呆警告，點擊後徹底清除包含教練排定的整個月所有預約資料。
- 會員專區 UI 質感升級：教練已約時段現在擁有獨立的立體凸版視覺效果。
- 按鈕手感統一化：清空按鈕繼承原本批量模式的互動視覺（Hover 凸起、Active 深陷凹版）。

### 🔧 修正 (Fixed)
- 解決當日預約客滿 17 小時造成的「最後一行消失」溢出遮擋問題，藉由為時間面板實裝 `max-height: 85vh` 與內部捲軸機制徹底解決。
- 解決連續七天皆排滿導致的 CSS Grid 左右溢出爆版：透過 `minmax(0, 1fr)` 與字串省略 (`text-overflow: ellipsis`) 強化網格極限拉扯能力。
- 新增底部 50px 畫面緩衝，防止手機瀏覽器的虛擬導航列遮擋介面最下緣。


## [V5.7] - 2026-04-12
### 🔧 修正 (Fixed)
- 修復連續排課卡控功能的 String/Boolean 型別錯判問題，確保阻擋機制能正常生效。
- 修復教練區 UI 誤導問題：在未選身分或當日客滿時，日曆格子不再產生凹版效果，游標維持預設，並實裝正確的點擊防呆提示（如「該日已無可操作時段」）。

### 🔒 資安 (Security)
- 移除登入失敗時 Toast 提示的明文測試密碼 (1/1)，防堵潛在的預設身分外洩。

### 🧹 清理 (Removed)
- 徹底清除過往遺留之診斷功能專用按鈕、資料庫聯通測試彈窗、JS 運算邏輯與臃腫之對應 CSS。
