# Changelog

## [V8.12] - 2026-04-14
### 系統邏輯升級
- **Dual-Mode 視角引擎**：實作「只顯示 X 課程」按鈕的聯動邏輯。關閉時為學員全域視角（顯示他教練預約為反灰），開啟時為精確過濾視角。
- **資料精確清空**：重構 `clearMonthData`。新邏輯僅移除執行者的預約標籤，不再誤刪同日其他人的課程，顯著提升資料安全性。
- **入口狀態重置**：強制進入教練專區時清空會員選取狀態，確保操作流程的一致性。

### UI/UX 視覺恢復
- **堂數統計回歸**：在教練圖例中補回該會員與各教練的「成交堂數」即時統計。
- **視覺重心校準**：修正未選取教練時的「全彩色」全局視角，避免初始狀態下的視覺疲勞。

### LINE 通知系統大翻新 (質感版)
- **狀態球導引**：使用 🟢(新增)、🔴(取消)、🟡(異動)、⚪(清空) 作為訊息識別開頭。
- **身分識別標記**：標題統一採「【姓名身分】課表【狀態】通知」格式，並精確區分學員與管理員操作。
- **詳盡清空報表**：執行清空月份時，若涉及教練課程，將自動條列式列出「受影響教練」與「詳細時段」。
- **排版優化**：加入質感橫向分隔線與時間項目符號(`•`)。

## [V8.11] - 2026-04-13
### 介面純化與教練專區修復
- **統計移除**：刪除「本月累計成交」統計橫幅與所有堂數標記，介面恢復純視覺預約表。
- **視角純化**：實作教練專區「過濾特定教練」時自動隱藏他教練課程的邏輯。
- **入口去學員化**：進入教練專區時強制清空學員選取狀態。
- **刪除功能強化**：手動刪除 User 邏輯與 saveData 整合，確保刪除後同步持久化。
