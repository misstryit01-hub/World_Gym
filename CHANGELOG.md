# 專案版本更新紀錄 (CHANGELOG)

## [V8.17] - 2026-04-14 (會員註冊視窗純淨化與語意極簡優化)
### 🚀 註冊介面質感升級
- **[UI] 介面純淨化**：移除註冊視窗所有英文標籤，將欄位名稱精簡為：帳號名稱、設定密碼、再次輸入密碼、推薦碼。
- **[UX] 語意極簡化**：移除欄位佔位符（placeholder）中的詳細規範提示，僅保留最直覺的引導文字。
- **[UI] 按鈕文字同步**：將註冊彈窗按鈕精簡為「取消」與「申請」，達成全站視覺一致性。

## [V8.16] - 2026-04-14 (忘記密碼引導與移動端佈局精煉)
### 🚀 安全引導與移動端佈局升級
- **[安全性] 忘記密碼樣式統一**：登入頁面底端新增「忘記密碼？」連結，視覺上與「立即申請」保持珊瑚紅高對比風格，並改為極簡的行政引導彈窗（聯繫管理員）。
- **[移動端] 教練區寬度適配**：精簡狀態標籤文字為「連排中」與「無卡控」，並縮小元件內距與行高，解決手持裝置文字疊合擠壓問題。
- **[UX] 引導文字清晰化**：教練身分選取提示全面強化，改為深黑色 (#333) 與 900 重磅字體，消除淺色系導致的辨識困難。

## [V8.15] - 2026-04-14 (教練控制列整合與模式視覺動態優化)
### 🚀 專業介面與動態視覺鎖定
- **[教練區] 控制列一體化**：年月導航文字絕對置中，並將「強制連續排課」狀態徽標內嵌至導航箭頭內部，提升視覺層次。
- **[教練區] 數據統計增強**：底部署員名單新增動態「x月總計」卡片，以醒目紅色數值呈現學員當月課表總和。
- **[會員區] 狀態圖示化**：為「快速複製模式」與「強制連續排課」導入動態 SVG 圖示（📄/📑 與 🔒/🔓），強化操作回饋感。
- **[修復] 模式連動歸零**：修正模式在「取消選取」或提交後未正確重置的重疊邏輯 Bug。
- **[UI] 精緻化視覺**：全站圖示風格統一為專業 SVG，確保線條粗細與視覺比例同步。

## [V8.14] - 2026-04-14 (全站確認視窗統一與視覺優化定版)
### 🚀 介面一致性與體驗優化
- **[UI] 全站視窗統一**：將所有 `openCustomConfirm` 標題統一為「確認送出嗎？」，並將按鈕標準化為「取消」與「確定」。
- **[UI] LINE 通知優化**：在環境偵測提示中導入 LINE 官方 SVG 圖示，強化品牌感。
- **[體驗] 語法精煉**：確保所有確認視窗均能準確捕捉並顯示當前操作的年月日或特定日期資訊。

## [V8.13] - 2026-04-14 (質感通知與身分識別完美版)
### 🚀 最終精煉與修復
- **[修復] 時間引擎防震化**：導入正則表達式提取技術，徹底根絕 LINE 通知中因標籤格式導致的時間 NaN 錯誤。
- **[優化] 身分純淨化**：移除教練專區中的所有「管理員」提示標籤，回歸最純粹的教練身分呈現。
- **[定版] 正式交貨**：所有視覺與邏輯細節均已通過校準，進入生產穩定狀態。
